# Product Website Readme

This website is a product website developed in Java. It includes the following features:

## Search
The user can search for products by brand, name, code, or all.

## Authentication
JWT authentication is used for login. Without a token, the user cannot switch to other pages.

## Images
Images of the products are placed in the resources folder of the product that you want to add.

## Reviews
Users can leave reviews/comments on products. The admin can approve or reject them.

## Technologies Used
- Java
- Spring Boot
- JWT Authentication
- MySQL

## How to Run the Application
1. Clone the repository to your local machine.
2. Open the project in your preferred IDE.
3. Configure the database connection in the `application.properties` file.
4. Run the application.
